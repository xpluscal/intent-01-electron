# Branding Guidelines

## Our Brand Personality

### Who We Are
- **Professional** but not corporate
- **Friendly** but not casual  
- **Smart** but not condescending
- **Optimistic** but not naive

### How We Talk
- Use simple words, avoid jargon
- Be encouraging about sustainability progress
- Focus on impact and results
- Speak directly to "you" not "users" or "customers"

### Typography

#### Font Family
**Inter** - Available from Google Fonts
```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
```

#### Font Weights
- Regular: 400 (Body text)
- Medium: 500 (Emphasized text)
- Semibold: 600 (Subheadings)
- Bold: 700 (Headlines)

#### Type Scale
- **Display**: 48px / 56px line-height (700 weight)
- **H1**: 36px / 44px line-height (700 weight)
- **H2**: 30px / 38px line-height (600 weight)
- **H3**: 24px / 32px line-height (600 weight)
- **H4**: 20px / 28px line-height (600 weight)
- **Body Large**: 18px / 28px line-height (400 weight)
- **Body**: 16px / 24px line-height (400 weight)
- **Small**: 14px / 20px line-height (400 weight)
- **Caption**: 12px / 16px line-height (400 weight)

### Logo Specifications
- Minimum width: 120px (digital), 1.5 inches (print)
- Clear space: Half the logo height on all sides
- Always use provided logo files, never recreate

### Spacing System
Use multiples of 4px:
- **4px**: Tight spacing (inside components)
- **8px**: Default spacing
- **16px**: Medium spacing (between related items)
- **24px**: Large spacing (between sections)
- **32px**: Extra large spacing (major sections)
- **48px**: Page margins on desktop

## Writing Style

### Tone of Voice
- **Clear**: Say what you mean, no fluff
- **Helpful**: Always provide next steps
- **Positive**: Focus on what's possible
- **Human**: Write like you're talking to a colleague

### Content Guidelines

#### Headlines
- Sentence case (not Title Case)
- Maximum 8 words
- Action-oriented when possible
- No period at the end

#### Body Copy
- Short paragraphs (3-4 sentences max)
- Use bullet points for lists
- Bold for emphasis, not italics
- One idea per paragraph

#### Button Text
- Start with a verb
- 2-3 words maximum
- Sentence case
- Examples: "Get started", "View report", "Add data"

### Examples

**Good**: "You've reduced emissions by 12% this month!"  
**Not**: "Monthly emission reduction metrics indicate a 12% decrease"

**Good**: "Let's set up your first data source"  
**Not**: "Initialize primary data integration workflow"

**Good**: "Something went wrong. Try refreshing the page."  
**Not**: "Error 500: Internal server error occurred"

## Quick Reference

### Do's
- Use exact color codes provided
- Stick to Inter font family
- Follow the 4px spacing grid
- Keep copy concise and friendly

### Don'ts
- Don't create new colors
- Don't use different fonts
- Don't center-align body text
- Don't use all caps (except buttons)

Remember: Consistency builds trust. When in doubt, refer back to these guidelines.
